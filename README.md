# 🐢 Turtle Crossing Game

A fun arcade-style game built using Python's `turtle` module. The objective is simple: guide your turtle safely across the road while avoiding incoming traffic. Each successful crossing increases the difficulty!

---

## 🎮 Gameplay

- Control the turtle using the **Up Arrow key**
- Avoid colliding with moving cars 🚗
- Reach the top of the screen to level up
- Game ends on collision

---

## 🚀 Features

- Object-Oriented Design (Player, CarManager, Scoreboard)
- Dynamic car generation with random colors and positions
- Increasing difficulty with each level
- Real-time collision detection
- Clean and minimal UI using Turtle graphics

---

## 🛠️ Project Structure
    ├── main.py # Game loop and event handling
    ├── player.py # Player (turtle) logic
    ├── car_manager.py # Car creation and movement
    ├── scoreboard.py # Score and game over display

📦 Requirements
Python 3.x
Standard library only (turtle, time, random)
